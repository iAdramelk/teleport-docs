---
title: Database Automatic User Provisioning 
description: Configure automatic user provisioning for databases.
version: '15.x'
---

Teleport can automatically create users in your database, removing the need for
creating individual user accounts in advance or using the same set of shared
database accounts for all users.

Currently, automatic user provisioning is supported for the following databases:

- [Self-hosted and AWS RDS PostgreSQL databases](/docs/database-access/auto-user-provisioning/postgres)
- [Self-hosted and AWS RDS MySQL databases](/docs/database-access/auto-user-provisioning/mysql)
- [Self-hosted and AWS RDS MariaDB databases](/docs/database-access/auto-user-provisioning/mariadb)
- [AWS Redshift databases](/docs/database-access/auto-user-provisioning/aws-redshift)
