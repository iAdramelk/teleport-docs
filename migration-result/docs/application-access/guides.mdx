These guides explain how to use the Teleport Application Service, which allows
your teams to connect to applications within private networks with fine-grained
RBAC and audit logging.

Manage access to internal applications:

- [Web App Access](/docs/application-access/guides/connecting-apps): How to access web apps with Teleport.
- [TCP App Access](/docs/application-access/guides/tcp): How to access plain TCP apps with Teleport.
- [API Access](/docs/application-access/guides/api-access): How to access REST APIs with Teleport.
- [Dynamic Registration](/docs/application-access/guides/dynamic-registration): Register/unregister apps without restarting Teleport.
- [Amazon Athena Access](/docs/application-access/guides/amazon-athena): How to access Amazon Athena with Teleport.
- [Amazon DynamoDB Access](/docs/application-access/guides/dynamodb): How to access Amazon DynamoDB as an application.
- [Application Access HA](/docs/application-access/guides/ha): How to configure the Teleport Application Service for high availability.
