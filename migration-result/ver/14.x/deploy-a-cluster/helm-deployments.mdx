## Helm deployment guides

These guides show you how to set up a full self-hosted Teleport deployment using
our `teleport-cluster` Helm chart.

- [Teleport Cluster](/ver/14.x/deploy-a-cluster/helm-deployments/kubernetes-cluster): Run a Teleport cluster in a Kubernetes cluster using
  the default configuration. This deployment is a great starting point to try a self-hosted
  Teleport with minimal resources.
- [HA AWS Teleport Cluster](/ver/14.x/deploy-a-cluster/helm-deployments/aws): Running an HA Teleport cluster in Kubernetes using an AWS EKS Cluster
- [HA GCP Teleport Cluster](/ver/14.x/deploy-a-cluster/helm-deployments/gcp): Running an HA Teleport cluster in Kubernetes using a Google Cloud GKE Cluster
- [Custom Teleport config](/ver/14.x/deploy-a-cluster/helm-deployments/custom): Running a Teleport cluster in Kubernetes with a custom Teleport config
