Teleport's reference guides provide comprehensive resources for configuring and
running Teleport.

- [Configuration](/ver/16.x/reference/config): All configuration options for the `teleport`
  binary.
- [CLI](/ver/16.x/reference/cli): Commands, flags, and arguments for Teleport's CLI tools.
- [Predicate Language](/ver/16.x/reference/predicate-language): The syntax for Teleport's
  predicate language, which is used to define RBAC policies and search for
  resources.
- [Audit Events](/ver/16.x/reference/audit): A reference of Teleport's audit events.
- [Helm Reference](/ver/16.x/reference/helm-reference): References for Teleport's Helm charts.
- [Networking](/ver/16.x/reference/networking): Ports that Teleport services listen on, plus
  other information about how Teleport services communicate.
- [Authentication](/ver/16.x/reference/authentication): Teleport's authentication connectors.
- [Signals](/ver/16.x/reference/signals): The signals you can use to control a Teleport
  binary.
- [Backends](/ver/16.x/reference/backends): Configuration options and required setup steps for
  Teleport's Auth Service storage backends.
- [Join Methods](/ver/16.x/reference/join-methods): A reference of Teleport join methods and the token resources that configure them.
- [Terraform Provider](/ver/16.x/reference/terraform-provider): Configuration options and
  resources for the Teleport Terraform provider.
- [Metrics](/ver/16.x/reference/metrics): All metrics available in Teleport.
- [Resources](/ver/16.x/reference/resources): Configuration resources you can apply in
  Teleport.

<Tip>
  This section contains references that apply to all Teleport services. For
  references that are relevant to a single Teleport service or use case, consult
  the appropriate section of the documentation:

  - [Teleport Application Service](/ver/16.x/application-access/reference)
  - [Teleport Database Service](/ver/16.x/database-access/reference)
  - [Teleport Desktop Service](/ver/16.x/desktop-access/reference)
  - [Machine ID](/ver/16.x/machine-id/reference)
  - [RBAC](/ver/16.x/access-controls/reference)
  - [Login Rules](/ver/16.x/access-controls/login-rules/reference)
</Tip>
